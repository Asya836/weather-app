# WEATHER

Proje react ile oluşturulmuş, hava durumu bilgileri axios ile çekilmektedir.

Şehir girildikten sonra aşağıdaki bilgilere ulaşılabilir:
- derece
- açıklama
- nem
- rüzgar
- hissedilen sıcaklık
